# Taxonomy of Terms and Definitions

## Status

This workstream has been **merged into the cross-WG AAIF Taxonomy / Landscape effort** and is no longer run as a standalone Security and Privacy WG workstream. The cross-WG effort builds common terminology, resolves conflicting definitions, and maintains decoupled data updated through PRs. Contributions are open to everyone; each working group provides delegates who actively review and help resolve terminology questions.

Junjie Bu chairs the cross-WG Taxonomy / Landscape workstream. See the [2026-06-09 WG meeting notes](../../meeting-notes/2026-06-09.md) for details.

## Purpose

Establish a shared taxonomy of security and privacy terms and definitions for agentic AI, carrying the SP WG's existing draft forward as a starting point into the broader cross-WG effort.
