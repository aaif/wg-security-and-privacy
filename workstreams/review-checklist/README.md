# Cross-WG Security and Privacy Review Checklist

## Purpose

Define a lightweight checklist and process for reviewing other AAIF working groups' deliverables through a security and privacy lens. The SP WG acts as an intersecting layer across AAIF; this workstream is what makes that role operational.

Initial review targets:

- **Identity & Trust** — agent identity, delegated authorization
- **Observability & Tracing** — behavioral anomaly detection, audit logging
- **Agentic Commerce** — payments security guardrails (fund draining, account leaks, payment-agent scope limits)
- All other AAIF WGs as deliverables emerge

The workstream targets an early v0 to socialize with other WG chairs so that security and privacy considerations are in their minds as they do their own work.
